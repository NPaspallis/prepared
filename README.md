# PREPARED App Framework

A framework for developing mobile educational apps with reusable and extensible component types

This software is Open Source as per the [MIT License](LICENSE).

For usage info, see the [docs](docs/README.md).

To evaluate the software technology used in this app we performed research using expert interviews. For these, we secured an [Approval from Cyprus Bioethics Committee](/docs/research/APPROVAL%20-%20ΕΕΒΚ%20ΕΠ%202025.01.38%20_Καθ.%20Νέαρχος%20Πασπαλλής%20_05.02.2025.pdf).
